# 更新日志 / Changelog

## v1.0.1 (2024-12-09)

🎉 **首个正式发布版本**

### ✨ 功能特点

- ✂️ **真正的剪切** - 在 Finder 中使用 ⌘X 剪切文件
- 📋 **智能粘贴** - 使用 ⌘V 移动文件到目标位置
- 🎯 **场景识别** - 自动区分文件选择和文本编辑状态
- 🔔 **可视化反馈** - 剪切/粘贴操作提供清晰的通知提示
- ⏱️ **超时保护** - 可自定义超时时间（1-30分钟）
- ⚙️ **设置界面** - 精美的毛玻璃偏好设置面板
- 🚀 **开机自启** - 支持开机自动启动
- 🔄 **自动更新** - 内置 Sparkle 自动更新
